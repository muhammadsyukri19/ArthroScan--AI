# Kaggle UAS Pack

Folder ini berisi file yang sebelumnya dibuat untuk eksperimen Kaggle:

- `notebook_kaggle_transfer_learning.ipynb`
- `train.py`
- `run_kaggle.sh`
- `requirements.txt`

Cara pakai singkat:

```bash
python train.py --data-dir /kaggle/input/<dataset-name> --batch-size 32 --epochs 20
```

Atau:

```bash
bash run_kaggle.sh
```
